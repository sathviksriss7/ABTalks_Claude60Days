📊 Week 1 — Data Foundations

🎯 Milestone: Confidently manipulate and visualize datasets using Python
Tools: Python, NumPy, Pandas, Matplotlib, Seaborn

🤖 Week 2 — Machine Learning Fundamentals

🎯 Milestone: Train, evaluate, and tune your first ML models
Tools: Scikit-learn, Jupyter Notebooks, Kaggle datasets

🔬 Week 3 — Applied ML & Feature Engineering

🎯 Milestone: Build a complete ML pipeline from raw data to predictions
Tools: Scikit-learn Pipelines, XGBoost, SHAP, Cross-validation

🚀 Week 4 — Deep Learning & AI Applications

🎯 Milestone: Build a neural network and deploy a simple AI-powered app
Tools: TensorFlow / Keras, Hugging Face, Streamlit, Google Colab

fast.ai (course.fast.ai)

🛠️ Capstone Projects

Week 1 Project
📈 Exploratory Data Analysis — Titanic or Zomato Dataset
Load a real dataset, clean nulls, create 5 visualizations, and write a summary of insights. Post on GitHub.
Week 2 Project
🏠 House Price Predictor
Train a Linear Regression and Random Forest on the Boston Housing dataset. Compare RMSE, tune hyperparameters, plot feature importances.
Week 3 Project
💳 Fraud Detection Pipeline
Build a full Scikit-learn Pipeline: imputation → encoding → scaling → XGBoost classifier. Handle class imbalance with SMOTE. Explain predictions with SHAP.
Week 4 Project
🤖 AI Mini-App with Streamlit
Build a Streamlit web app that takes user input and returns predictions from your Week 3 model — or uses a Hugging Face sentiment analysis model. Deploy free on Streamlit Cloud.

🏆 Final Outcome — What You'll Have After 30 Days

  • Strong command of Pandas, NumPy, and Matplotlib for data work
  • Ability to train, evaluate, and tune ML models with Scikit-learn
  • Understanding of core algorithms: Linear/Logistic Regression, Random Forest, XGBoost
  • Complete ML pipeline skill — from raw data to deployed predictions
  • Exposure to Deep Learning concepts and Hugging Face AI models
  • A GitHub portfolio with 4 real data science projects
  • A live Streamlit app you can share with anyone
  • Foundation to tackle Kaggle competitions or a DS/AI job prep track
